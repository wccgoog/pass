 ## 项目安装
 #### 第一步
 安装后台代码
 ```
 https://git.dev.tencent.com/xiaoliuxing/jiangbeizhengwu_new.git
 ```
 #### 第二步
 安装TP框架与项目用到的扩展
 ```
 composer install
 ```
 ### 第三步
 导入数据
 ```
 数据库导入项目目录下的jiangbeizhengwu.sql文件
 ```
 **注意本项目表数据都是使用`utf8mb4`的**

 ### 第四步
 修改**application/config.php**和**application/database.php**两个配置文件

 ### 第五步
 访问 <em>/admin</em> 登录后台
 帐号：admin 
 密码：不变
